## FAQ

**Q: Why now?**  
A: (1) Monad mainnet launching, (2) Enterprise focus on AI agents + governance, (3) Blockchain regulation maturing (regulators want this)

**Q: Who's the competition?**  
A: No direct competitor. Closest are: (1) Traditional devtools (no governance), (2) Traditional security tools (no blockchain), (3) Blockchain devtools (no governance or security). We're building a new category.

**Q: How do we win against bigger players?**  
A: Speed + community. We move fast, ship MVPs in weeks, listen to customers. Bigger players move slowly.

**Q: What if integration is harder than expected?**  
A: We have a fallback: deploy CyberSecurity-AI as a separate service (HTTP calls, not MCP embedding). Adds latency but unblocks team.

**Q: What if sales cycles are longer than 16 weeks?**  
A: We focus on freemium tier to build volume while hunting enterprise deals. Virality in crypto community can carry us.

**Q: Do we need funding?**  
A: To accelerate, yes. But we can bootstrap with customers (pay for premium features). Pre-sales can fund development.

**Q: What's the off-ramp if this doesn't work?**  
A: MonadBuilder+ and CyberSecurity-AI stand alone. Both have independent value. Worst case, we sell each separately or as components to other platforms.

## Decision Tree (What to Do If...)

### If we hit $50K MRR earlier than expected (Good Problem)
→ Hire 2 more engineers to build features faster  
→ Hire sales engineer to manage expansion  
→ Start Tier 3 (enterprise) focus

### If integration is taking longer than expected (Medium Problem)
→ Pivot to separate service model (HTTP calls)  
→ Ship integration with reduced feature set (MVP)  
→ Parallelize: sales team can launch while integration finishes

### If pilots aren't converting (Bad Problem)
→ Analyze: is it product fit or sales process?  
→ Product fit: iterate based on pilot feedback  
→ Sales process: change messaging, target different segment  
→ Last resort: pivot to freemium focus, build volume

### If we get acquisition offer (Good Problem)
→ Evaluate based on: strategic fit, team retention, customer impact  
→ Don't sell if we're on trajectory to $100M ARR independently  
→ Only sell if it accelerates our vision (e.g., Monad Foundation acquires us)
