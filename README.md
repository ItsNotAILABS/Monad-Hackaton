# MonadBuilder+ · THESIS

> **Build the application. Govern the intelligence. Execute on Monad.**

MonadBuilder+ and THESIS are one integrated product stack for creating, operating, and governing applications on Monad.

- **MonadBuilder+** is the human-facing application studio: describe an idea, generate a full dApp, refine it visually, connect or create wallets, inspect live chain state, publish projects, and learn the system through use
